# tetris-game
- en [English](README.md)
- zh_CN [简体中文](README.zh_CN.md)

这是一个简单的俄罗斯方块游戏，使用 Java 完成，运行在 IntelliJ Platform 上，请前往[插件市场](https://plugins.jetbrains.com/plugin/23860-tetris-game?noRedirect=true)获取安装包。

## 操作
游戏支持自定义按键，默认设定如下：  
- 重新开始：```R```
- 硬降：```空格```
- 软降：下箭头⬇️
- 左移动：左箭头⬅️
- 右移动：右箭头➡️
- 向右旋转：上箭头⬆️
- 向左旋转：```Z```
- 暂存：```C```
